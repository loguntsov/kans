Гидравлический расчет ливневой канализационной сети.

Логунцов С.В., Логунцов В.Ф. 2002-2004.


Программа позволяет произвести гидравлический расчет согласно ГОСТ ливневой канализационной сети с применением методов численного моделирования.

Воможности:
1. Определение диаметра, уклонов, наполнения и отметок отчастков канализационной сети.
2. Расчет и определение местоположения станций перекачки сточных вод.
3. Удобный интерфейс для редактирования параметров сети
4. HTML-отчет о с результатами расчета.

Дополниительно включены презентационные материалы и описание методики расчета.

------

В память о Логунцове Виталии Федоровиче (1953 - 2010)
